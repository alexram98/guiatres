Etiquetas para pagina web
Etiquetas Descripcion
(!DOCTYPE html) Informa a los navegadores que a continuación van a leer HTML5
(html) Informa a los navegadores que en su interior es dónde vamos a encontrar el HTML5
(head) Es la cabecera del documento web.
(title) Informa a los motores de búsqueda cual es el contenido de la página web.
(body) Es el cuerpo del documento web.
(a) Destinos:
  • archivos (página web, imagen o cualquier otro tipo de archivo)
  • directorios
  • elementos dentro de una página web
    		•
    (header) Está pensada para agrupar el contenido inicial de una página web
    (nav) Está pensada para agrupar los enlaces a otras páginas o los enlaces a los apartados de la propia página.
    (section) Está pensada para agrupar los apartados del contenido principal del documento
    (article) Está pensada para agrupar el contenido de la página que forma una unidad en sí misma desde el punto de vista temático.
    (aside) Está pensada para agrupar contenido secundario y tangencial al contenido al que acompaña
    (footer) Está pensada para contener información general sobre el documento, información que se suele poner al final del documento.
    (address) Está pensada para contener información de contacto del autor de la información (dirección de correo, teléfono, etc.)
    (div) Define una división.
    (main) Se utiliza para agrupar el contenido principal de una página, sin incluir los elementos comunes a otras páginas
    (table) Definición de una tabla.
    (th) Definición de cabecera de tabla
    (tr) Define una fila en la tabla.
    (td) Define una celda en la tabla.
    (caption) Define el nombre o título de la tabla
    (colgroup) Especifica un grupo de una o más columnas para aplicar formato.
    (col) Especifica las propiedades de una columna de las columnas definidas en un elemento colgroup
    (thead) Define la cabecera de la tabla.
    (tbody) Define el cuerpo de la tabla
    (tfoot) Define el pie de la tabla
